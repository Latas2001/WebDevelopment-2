# Terminal Commands

### List of Commands:

###### Current directory:
```
pwd
```

###### Change directory:
```
cd

// go one level up in directory tree
cd ..

// go to root
cd ~
```

###### Create a directory:
```
mkdir
```

###### List directory contents:
```
// shows all content
ls
```

```
// shows all content with details
ls -a
```

###### Create a file:
```
touch <file_name>
```

###### Read file contents:
```
cat <file_name>
```
